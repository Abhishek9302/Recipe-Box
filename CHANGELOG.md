### Added - ABH-10 Recipe Box
- Built a full-stack Recipe Box experience with a Next.js frontend and an Express + TypeScript backend
- Added recipe browsing, search, category filtering, difficulty filtering, and a recipe detail view
- Added recipe creation, editing, and deletion flows
- Added JWT-based signup/login and authenticated favorites support
- Added PostgreSQL schema and seed data for users, categories, recipes, ingredients, and favorites
- Added `/health` service endpoint for API readiness checks
- Added deployment and implementation documentation for release handoff
